
# Wallpapers for AerynOS

Welcome to this wallpaper repository!
These landscape and nature photographs were taken over the last few years with a DSLR (Canon EOS 600D) and a Drone (DJI Mavic 2 Pro).

![Moss Sprouts](src/moss-sprouts.jpg)

This collection is primarily curated and contributed for the [**AerynOS**](https://aerynos.com/) project, offering them as candidates for desktop wallpapers. However, thanks to the open-source license, anyone in the community is free to use them!
